Enfasis de texto

Para enfatizar el texto podemos usar negritas y cursivas,
para las negritas se encierra el contenido en la etiqueta "strong"
para las cursivas se utiliza la etiqueta "em"

Listas

Para crear listas no ordenadas se utiliza la etiqueta "ul"
y para las listas ordenadas se utiliza la etiqueta "ol"

Lista no ordenada (ul)

  • Primera cosa
  • Segunda cosa
  • Tercera cosa

Lista ordenada (ol)

  1. Primera cosa
  2. Segunda cosa
  3. Tercera cosa

Imágenes

Para colocar imagenes debemos utilizar la etiqueta "img",
posteriormente debemos utilizar el atributo "src" y agregarle el valor mediante el simbolo "=".
Podemos agregar una descripción a la imagen por si falla el alojamiento de la misma utilizando el atributo "alt"
y podemos modificar su tamaño mediante el atributo "width"

CSS

El CSS es el lenguaje de programación que nos permite dar el estilo a nuestra página web
Para esto debemos utilizar la etiqueta "style" dentro de la cabeza de nuestro código html.
Posteriormente utilizaremos selectores para elegir que parte de nuestro código debe ser estilizada.
Los selectores van seguido de llaves {} y dentro de esas llaves va el código para cambiar el color u otros aspectos.

Seleccionar por ID

Para seleccionar por id, debemos poner "id" dentro de la etiqueta de apertura que queremos seleccionar,
y asgnarle un valor con el signo "=", debemos llamar en style al id con el signo "#".

Seleccionar por Clase

para seleccionar por clase, debemos poner "class" dentro de la etiqueta de apertura que queremos seleccionar,
y asignarle el valor con el signo "=", debemos llamar en style a la clase con el signo ".".

Vínculos en HTML

Para generar vínculos en html debemos utilizar la etiqueta "a",
posteriormente dentro de la apertura y cierre de dicha etiqueta escibimos el texto visible en nuestra página
y generamos el vínculo dentro de la etiqueta de apertura con el atributo "href=".
Si queremos que el enlace se abra en una nueva pestaña debemos agregar antes del "href" el atributo "target="_blank""
Ejemplo: Aprende html en KhanAcademy

Vínculos internos

Para generar vínculos internos debemos poner la etiqueta "a" y colocar dentro de la apertura el atributo href="#...",
de esta forma el navegador nos llevará donde el ID que tengamos en el href.
Ejemplo:Vuelve al inicio de la página

Tablas

Para crear tablas debemos utilizar la etiqueta "table".
Para agregar areas de nombres de columnas debemos utilizar la etiqueta "thead".
Para agregar una fila de encabezados debemos utilizar la etiqueta "tr".
Para agregar encabezados a las celdas utilizamos la etiqueta "th".
Para agregar datos a la tabla debemos utilizar la etiqueta "tbody" bajo el cierre de "thead".
Volvemos a agregar filas con la etiqueta "tr"
Agregamos datos a las celdas con la etiqueta "td"

Cambiar la fuente

Para cambiar la fuente debemos ocupar el atributo "font-family" detro de nuestro style en css.
Algunas familias de fuentes genericas son las siguientes: sans-serif, serif, fantasy, monospace y cursive.

Cambiar el tamaño de la fuente

Para cambiar el tamaño de la fuente debemos ocupar el atributo "font-size" dentro de nuestro style en css.
Podemos utilizar 2 tipos de atributos por pixel "Xpx" o por equivalencia "Xem".
